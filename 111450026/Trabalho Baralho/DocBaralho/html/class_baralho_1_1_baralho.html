Detailed Description

Classe que representa o baralho. O baralho conta com dois montes: Monte principal: monte inicial do baralho. O monte que sofre o corte e onde se compram as cartas. Monte de descarte: monte para vão as cartas descartadas pelos usuários do baralho.
